# 📦 Sistema de Controle Operacional KC

Projeto desenvolvido com base em necessidades reais do ambiente de trabalho, com foco no controle de estoque e operações logísticas.  
Atualmente, o sistema encontra-se em fase de desenvolvimento contínuo, com melhorias sendo implementadas gradualmente.

---

## 🚀 Funcionalidades

- Registro de entrada de materiais
- Controle de saída de kits
- Validação para evitar inconsistências de estoque
- Organização de dados operacionais
- Estrutura modular do sistema

---

## 🧠 Objetivo

Aplicar conceitos de programação para resolver problemas reais de controle operacional, aumentando a eficiência e reduzindo erros.

---

## 🛠 Tecnologias utilizadas

- Python
- SQLite
- Estrutura modular (separação por camadas)

---

## ▶️ Como executar

```bash
pip install streamlit
python app.py

```
